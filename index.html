API Integration Example

API Description - Ella L'Esperance Assignment 8, API Fun!

This example uses the Spoonacular API to create a weekly meal plan.

I found this example using the list of public APIs to create a weekly meal plan, listed here.

The API endpoint used is: https://api.spoonacular.com/mealplanner/generate

Parameters needed were an API Key that was provided with a sign up to the API. This was free to do, and my URL look like this:
https://api.spoonacular.com/mealplanner/generate?apiKey=7c7f3534ec1e4636b081d4a9332f9a35

Potential applications for this API include:

  • Create a weekly meal plan schedule for someone to follow
  • Give potential recipe ideas for each meal of the week
